# padi-midi-cv
Glue code that runs on a Padi IoT stamp, interprets midi messages, spits them out to a screen and converts them to a CV signal for modular synth modules.

## Connections
Are documented in the code comments

## Getting the code running on a PADI IoT stamp
Use the package here: https://github.com/gtalusan/PADIIoTStamp

It's a bit broken so you might want to use the fixes I have posted in my issue here: https://github.com/gtalusan/PADIIoTStamp/issues/4
